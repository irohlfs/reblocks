API

  • [function]
    reblocks/response:add-header
    name value

    Use this function to add a HTTP header:

    (add-header :x-request-id 100500)
  • [function]
    reblocks/response:add-retpath-to
    uri &key (retpath (get-uri))

    Adds a "retpath" GET parameter to the giving URL.

    Keeps all other parameters and overwrites "retpath" parameter if it is already exists in the URL.

    By default, retpath is the current page, rendered by the reblocks. This is very useful to redirect user to login page and return him to the same page where he has been before.
  • [class]
    reblocks/response:response
  • [reader]
    reblocks/response:get-code
    (:code = 200)

    HTTP status code to return in response to request.

    By default, this slot will be set to 200.
  • [reader]
    reblocks/response:get-content
    (:content = "")

    A string with a content of the response.
  • [reader]
    reblocks/response:get-content-type
    (:content-type = (get-default-content-type-for-response))

    HTTP content type to return in response to request.

    By default, have text/html value for usual requests and application/json for AJAX requests.
  • [reader]
    reblocks/response:get-custom-headers
    (:custom-headers = nil)

    Custom HTTP headers of request.

    By default, this slot will be set to 200.
  • [function]
    reblocks/response:get-headers
    response
  • [reader]
    reblocks/response:get-response
    (:response)
  • [function]
    reblocks/response:immediate-response
    content &key (content-type (get-default-content-type-for-response)) (code 200) (headers \*custom-headers\*) (condition-class 'immediate-response)

    Aborts request processing by signaling an immediate-response and returns a given value as response.

    HTTP code and headers are taken from CODE and CONTENT-TYPE.
  • [condition]
    reblocks/response:immediate-response
  • [function]
    reblocks/response:make-response
    content &key (code 200) (content-type (get-default-content-type-for-response)) (headers \*custom-headers\*)
  • [function]
    reblocks/response:make-uri
    new-path

    Makes a new URL, based on the current request's URL.

    If new-path can be absolute, like /logout or relative, like ./stories.

    Also, it can contain a query params like /login?code=100500
  • [function]
    reblocks/response:send-script
    script &optional (place :after-load)

    Send JavaScript to the browser. The way of sending depends on whether the current request is via AJAX or not.

    Script may be either a string or a list; if it is a list it will be compiled through Parenscript first.
